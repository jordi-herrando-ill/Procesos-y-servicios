1. Descargar y Mostrar Página Web

Descripción

Esta aplicación Java utiliza la funcionalidad de futuros para sincronizar la descarga y visualización de una página web. El usuario proporciona una URL, la aplicación descarga la página y muestra su contenido.
Instrucciones de Uso

Ejecuta la aplicación y sigue las instrucciones para ingresar la URL. Una vez ingresada, la aplicación descargará la página y mostrará su contenido.



2. Comprimir y Mover Archivo

Descripción

Esta aplicación Java permite al usuario comprimir una carpeta o archivo en formato ZIP y luego mover el archivo ZIP resultante a una ubicación específica. Utiliza futuros para coordinar estas dos tareas.
Instrucciones de Uso

Ingresa las rutas de origen y destino cuando se te solicite. La aplicación comprimirá la carpeta/archivo y moverá el archivo ZIP resultante a la ubicación especificada.



3. Descargar y Comprimir Páginas Web

Descripción

Esta aplicación Java descarga el contenido de 10 páginas web proporcionadas por el usuario, luego comprime estos contenidos en un archivo ZIP. La implementación utiliza futuros para gestionar la descarga simultánea de múltiples páginas.
Instrucciones de Uso

Proporciona las URLs de las 10 páginas web cuando se te solicite. La aplicación descargará los contenidos y los comprimirá en un archivo ZIP llamado "paginas_web.zip".
